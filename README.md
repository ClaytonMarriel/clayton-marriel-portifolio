# Clayton Marriel | Portfolio

A modern, responsive and multilingual developer portfolio built with **Angular 21**, focused on presenting my professional experience, corporate projects, technical stack and contact information in a clean and polished way.

## Preview

This portfolio was designed to communicate a strong professional image, highlighting:

- Senior frontend positioning
- Corporate experience
- Featured client projects
- Multilingual support
- Responsive UI
- Clean and modern visual identity

## About the project

This is my personal portfolio website, created to showcase my experience as a **Senior Frontend Engineer** with focus on **Angular, TypeScript and React**.

The application was built with a modern frontend structure and includes sections for:

- Hero / professional presentation
- About
- Professional experience
- Featured corporate projects
- Tech stack
- Contact

The goal is to present my professional profile in a way that is visually strong, technically clear, and aligned with enterprise opportunities.

## Main features

- **Modern UI**
- **Fully responsive layout**
- **Multilingual support**: Portuguese, English and Spanish
- **Featured corporate projects section**
- **Expandable projects list** with “View More”
- **Professional contact section**
- **Custom favicon and visual identity**
- **Angular SSR structure**
- **Tailwind CSS integration**
